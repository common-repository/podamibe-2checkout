== Description ==

<strong>Podamibe 2Checkout</strong> is a FREE WordPress plugin that help you to pay easily. It also allows you to add/display the pay button on any post or page or sidebar easily. The Podamibe 2Checkout plugin is responsive and feature rich which is built up to simplify the user needs.

= Free Features: =

* <strong>Shortcode Based</strong>
* <strong>Multilanguage Ready</strong>
* <strong>Credit Card Payment Option</strong>
* <strong>PayPal Payment Option</strong>

== Installation ==

1. Unzip podamibe-2checkout.zip.
2. Upload all the files to the /wp-content/plugins/podamibe-2checkout.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on 2Checkout in Wordpress left admin menu and click on Settings.

= How can I display the Podamibe 2Checkout payment options? =
Use this for "[P2Checkout amount="give your pay amount here" orderid="give order id here"]" shortcode to display in content. For template use &lt;?php echo do_shortcode('<strong>[P2Checkout amount="give your pay amount here" orderid="give order id here"]</strong>');?&gt;.
